
# phpunit telepítése

```
composer require --dev phpunit/phpunit
```

## phpunit config file generálása:

Telepíteni kell hozzá globalisan is a phpunit-ot (windows-on)

```
composer global require phpunit/phpunit
```

Majd:

```
phpunit --generate-configuration
```
